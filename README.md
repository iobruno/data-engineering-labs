# Data Engineering Labs

A hands-on collection of data engineering projects. The module structure is inspired by the [Data Engineering Zoomcamp](https://github.com/DataTalksClub/data-engineering-zoomcamp) from [DataTalks.Club](https://datatalks.club/). All implementations are original work.

Refer to the modules below for covered topics and tools.
